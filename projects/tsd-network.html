About the Project

When I started with Triangle Site Design, all the desktop computers had wireless cards.there were 2 wireless routers serving all the desktop and laptops. Cisco switch whose limit with firewall protection was 50 mbps

  • Desktop computers
  • Cisco Meraki MX60
    • Stateful firewall throughput 100 Mbps Advanced security throughput 50 Mbps Recommended use cases - 20 users
Computers network connectivity were slow. Updating the drivers on the wireless cards helped. But computers were still maxing out at 50 mpbs Bottlenecking and latency - all traffic was being filter through 2 access points who switching capability was suspect Expandability was limited - we later added IP phones
