Demo Magnific Popup in Jekyll Website

Published: Jul 7, 2020
Category: Notes
Tags:  demo  gallery  jekyll  magnific-popup 

The following demonstrates integrating Magnific Popup in to a Jekyll generated website. I was digging around for a simple photo gallery rendering framework to use in my website and this was one that I came across. I wanted something to take in to account loading smaller sized thumbnail images and then select expanding to view larger size images. I ended up settling on this because it was simple, quick and easy to add and it worked well. Source can be in the github website repository. Also, here is a example of where I used it in a writeup.
